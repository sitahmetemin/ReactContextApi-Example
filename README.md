
## Nedir?

Bu uygulamada Bootstrap ve React Context Api kullanılmış ve örnek olarak kullnılması amaçlanmıştır. 

Kullanabilmek için: 

### `npm i`
komutunu kullanınız. Uygulamayı başlatmak için

### `npm start`
komutunu kullanabilsiniz.
